## Usage

```shell

git clone https://github.com/19Qingfeng/pages  克隆仓库

yarn 安装依赖

yarn dev 启动开发环境

yarn build 构建生产环境包

yarn build-all  构建所有页面生产环境包
```

推荐使用`yarn`基于项目`yarn.lock`替代`npm`进行项目环境安装
